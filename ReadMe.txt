To run the project, write the command: python Project.py /path/to/model.keras /path/to/the/project_folder/
If the model is already trained, use the --no-train option writing: python Project.py --no-train /path/to/model.keras /path/to/the/project_folder/.
Be careful! If /path/to/model.keras exist and the --no-train option is not specified then it will be overridden.
